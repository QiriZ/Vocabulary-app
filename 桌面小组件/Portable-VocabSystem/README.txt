Vocabulary Learning System - Portable Version 
-------------------------------------------- 
1. Double-click "start-vocab-system.bat" to run the program 
2. In any application, select text and press Alt+Space to quickly add vocabulary 
3. View history records that only show words uploaded by the current account 
4. Account name will correctly be sent to the "Identifier" field in Feishu table 
5. You can copy the entire folder to use on other computers
